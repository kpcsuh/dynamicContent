Problem Statement: Using legacy Q-Global product bundles in Angular App

This proof of concept (POC) is the work to prove existing Q-Global bundles in modern Angular (4) framework. This app loads existing Q-Global bundles in Angular application. This POC is required to prove that bundle migration is possible in Angular apps without rewriting them from scratch.
